# Franzininho WiFi

A placa Franzininho WiFi é uma placa de desenvolvimento baseada no módulo ESP32-S2-WROOM.

## Características

- **Módulo ESP32-S2-WROOM**
  - Microcontrolador Xtensa® single-core 32 bits LX7 operando em até 240 MHz
  - Memória:
    - 128 KB de ROM
    - 320 KB de SRAM
    - 16 KB de SRAM RTC
    - 4 MB de Flash
  - Wi-Fi 802.11 b/g/n

- **Interfaces**
  - GPIO, SPI, LCD, UART, I2C, I2S
  - Câmera, IR, contador de pulsos, LED PWM
  - TWAI (CAN), USB 1.1 OTG
  - ADC, DAC, sensor de toque, sensor de temperatura interno

- **Pinagem**
  - 40 pinos organizados em 2x20 headers (passo de 2,54 mm)
  - 35 GPIOs disponíveis
  - Compatível com protoboard

- **LEDs**
  - 2 LEDs de uso geral

- **Botões**
  - 1 × Reset
  - 1 × Boot

- **USB**
  - Conector Micro USB (USB 1.1 OTG)

- **Alimentação**
  - 5V via conector USB
  - 5V e GND via pinos
  - 3,3V e GND via pinos

- **Suporte de Programação**
  - ESP-IDF
  - Arduino
  - CircuitPython
  - MicroPython
